本插件提供三个功能：
1) coder：
view => 显示粘贴板文本
encode => 粘贴板文字的 base64 加密
decode => 粘贴板文本的 base64 解密

2) position recurser:
locate => 跳转到 pos 指向的滚轮位置（阅读位置）
switch to ... => 打开一个跳转阅读位置的 modal
add position => 新增一个
undo => 跳转到上一个 pos 指向的滚轮位置
redo => 跳转到下一个 pos 指向的滚轮位置
